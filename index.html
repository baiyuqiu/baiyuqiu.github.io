Hexo的使用

发表于 2022-04-26 更新于 2022-05-15 分类于 随笔

基础使用（命令）

hexo s

作用：启动本地服务器，只能用于预览在_posts目录下的主题，默认地址： http://localhost:4000/。

  • hexo s 是 hexo server 的缩写，命令效果一致；
  • 对于主题或_config.yml的修改，需要重启本地服务器后才能预览效果；
  • 如果预览草稿中的主题，需要执行hexo s --draft；
